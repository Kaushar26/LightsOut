# F1Hub – Driver Manager 🏎️

A Full Stack MERN application to manage Formula 1 driver data.
Users can add, update, view, and delete drivers along with their team and points.

---

## Features

* Add new drivers
* View all drivers
* Update driver details
* Delete drivers
* Data stored in MongoDB
* REST API built using Express
* React frontend interface

## Installation

Clone the repository

```
git clone https://github.com/Kaushar26/LightsOut.git
```

### Run Backend

```
cd backend
npm install
node index.js
```

### Run Frontend

```
cd frontend
npm install
npm run dev
```

Then open:

```
http://localhost:5173
```
